What is Arduino Uno?

The Arduino Uno is an open-source microcontroller board based on the ATmega328P chip. It's the go-to starting point for anyone learning electronics and embedded systems. It has 14 digital I/O pins, 6 analog inputs, a USB connection for programming, and runs on 5V ‚Äî making it safe and forgiving for beginners.

What is ESP32?

The ESP32 is the big brother of the ESP8266. It has a dual-core 240MHz processor, built-in Wi-Fi AND Bluetooth, 36 GPIO pins, 2 DAC outputs, hall sensor, capacitive touch pins, and much more. It's the chip used in professional IoT products and handles complex tasks like camera feeds, audio, and BLE beacons.
ProcessorXtensa LX6 Dual-Core

Wi-Fi802.11 b/g/n

GPIO Pins36

Touch Pins10 capacitive touch
Note: ESP32 also runs at 3.3V logic. Some ADC2 pins don't work when Wi-Fi is active. Touch pins (T0‚ÄìT9) are great for touchless switches.

Servo motors rotate to a precise angle (0‚Äì180¬∞). Controlled by a single PWM signal. 3 wires: VCC (red), GND (brown), Signal (orange). The SG90 is the most common and cheapest servo for beginners.

DC motors spin freely in both directions ‚Äî perfect for wheels. The L298N module lets you control direction and speed using PWM. It can drive 2 motors simultaneously, and has a built-in 5V regulator.

Building a Robot from Scratch

A robot is just a combination of all the components you've learned: a microcontroller (brain), motors (movement), sensors (perception), and power supply. Here's the typical stack for a beginner robot car.

üß† Brain

Arduino Uno or ESP8266 runs your program and coordinates all parts.

‚öôÔ∏è Movement

4√ó DC motors + L298N driver or L293D shield for drive control.

üëÅÔ∏è Perception

HC-SR04 for obstacle avoidance, IR sensors for line following.

üîã Power

18650 battery pack or 4√ó AA batteries. Always use a regulated supply.

üèóÔ∏è Chassis

Acrylic or aluminum 4WD chassis kit with wheels and standoffs.

üì° Remote

Bluetooth (HC-05), IR remote, or Wi-Fi for wireless control.

// 4WD Robot ‚Äî Obstacle Avoider using HC-SR04
// L298N: ENA=5, IN1=7, IN2=6, IN3=4, IN4=3, ENB=2 // HC-SR04: TRIG=9, ECHO=10 #define TRIG 9 #define ECHO 10 #define IN1 7 #define IN2 6 #define IN3 4 #define IN4 3 long getDistance() { digitalWrite(TRIG, LOW); delayMicroseconds(2); digitalWrite(TRIG, HIGH); delayMicroseconds(10); digitalWrite(TRIG, LOW); return pulseIn(ECHO, HIGH) * 0.034 / 2; } void forward() { digitalWrite(IN1,HIGH);digitalWrite(IN2,LOW);digitalWrite(IN3,HIGH);digitalWrite(IN4,LOW); } void turnLeft() { digitalWrite(IN1,LOW);digitalWrite(IN2,HIGH);digitalWrite(IN3,HIGH);digitalWrite(IN4,LOW); } void stopBot() { digitalWrite(IN1,LOW);digitalWrite(IN2,LOW);digitalWrite(IN3,LOW);digitalWrite(IN4,LOW); } void setup() { pinMode(TRIG,OUTPUT); pinMode(ECHO,INPUT); pinMode(IN1,OUTPUT); pinMode(IN2,OUTPUT); pinMode(IN3,OUTPUT); pinMode(IN4,OUTPUT); } void loop() { long dist = getDistance(); if (dist > 20) { forward(); } else { stopBot(); delay(300); turnLeft(); delay(400); } }
